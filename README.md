## Rocks

#### `node svg-compile.js`

To transform svg files into react components you must follow these steps

    $ cd ./config
    $ node svg-compile.js

svg files must be at 
    
    rootFolder/assets/svg
    
generated file will be at

    rootFolder/src/assets/svg/index.js

### Run
  
    $ npm start  

#### `Run IOS`

    $ npm run ios
    
#### `Run Android`

    $ npm run android


### Build

#### Android

    $ npm run build:android  
    
#### IOS

    $ npm run build:ios
